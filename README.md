# What is this
Hey so I built a website for a YSWS called perhiphral. In this ysws you design perhipherals and get sent the parts to turn it into reality. Note*This is just a landing page


so how it would work is that you would go on the website and design your own peripheral device using a cad software and get the parts needed for it shipped to you.

Or you could ship your project to get scraps to get on the shop!
